Criar um site envolve vários passos e é fácil esquecer alguma coisa importante. Para te ajudar, preparamos um **checklist completo** com tudo o que você precisa fazer para ter um site profissional e funcional.

Use este guia como referência antes, durante e depois da criação do seu site.

## Antes de criar o site

### Planejamento

- [ ] Defina o **objetivo principal** do site (gerar contatos, vender, informar)
- [ ] Liste os **serviços ou produtos** que serão apresentados
- [ ] Identifique seu **público-alvo** (quem são seus clientes ideais)
- [ ] Pesquise os **sites dos concorrentes** para referência
- [ ] Defina as **páginas** que o site terá (Início, Sobre, Serviços, Contato)

### Domínio e hospedagem

- [ ] Escolha o **nome do domínio** (curto, fácil de lembrar)
- [ ] Verifique a **disponibilidade** no Registro.br
- [ ] **Registre o domínio** (.com.br)
- [ ] Escolha a **forma de criação** (Linkfl, WordPress, agência)
- [ ] Contrate a **hospedagem** (se necessário — no Linkfl já está incluída)

## Durante a criação do site

### Conteúdo

- [ ] Escreva o **título principal** da página inicial (claro e direto)
- [ ] Crie a **descrição dos serviços** (com benefícios)
- [ ] Escreva a **página Sobre** (história, formação, diferenciais)
- [ ] Reúna **depoimentos** de clientes satisfeitos
- [ ] Prepare **perguntas frequentes** (FAQ)
- [ ] Defina os **textos dos botões** de ação (CTAs)
- [ ] Escreva **meta titles** e **meta descriptions** para SEO

### Imagens

- [ ] Tire ou selecione **fotos profissionais** (do espaço, equipe, trabalhos)
- [ ] **Otimize as imagens** (comprima para não pesar o site)
- [ ] Prepare o **logo** em alta resolução
- [ ] Tenha o **favicon** (ícone pequeno que aparece na aba do navegador)

### Design e funcionalidades

- [ ] Site **responsivo** (funciona bem no celular)
- [ ] **Botão do WhatsApp** flutuante em todas as páginas
- [ ] **Formulário de contato** funcional com notificação por email
- [ ] **Telefone clicável** (ao tocar, faz a ligação)
- [ ] **Endereço** com link para o Google Maps
- [ ] **Horário de atendimento** visível
- [ ] **Cores e fontes** consistentes com a identidade visual
- [ ] **Menu de navegação** simples e intuitivo

### SEO

- [ ] **Título SEO** (title tag) otimizado em cada página
- [ ] **Meta description** atrativa em cada página
- [ ] **URLs amigáveis** (sem códigos ou números)
- [ ] **Heading tags** corretas (H1, H2, H3)
- [ ] **Palavra-chave principal** nos títulos e textos
- [ ] **Alt text** nas imagens (descrição para o Google)
- [ ] Site com **HTTPS/SSL** ativado

## Após publicar o site

### Verificações técnicas

- [ ] Teste o site no **Chrome, Safari e Firefox**
- [ ] Teste no **celular e tablet**
- [ ] Verifique se todos os **links funcionam**
- [ ] Confirme que o **formulário envia** corretamente
- [ ] Teste o **botão do WhatsApp**
- [ ] Verifique a **velocidade** do site (use o Google PageSpeed)

### Google e ferramentas

- [ ] Cadastre o site no **Google Search Console**
- [ ] Instale o **Google Analytics** (para acompanhar visitas)
- [ ] Cadastre a empresa no **Google Meu Negócio**
- [ ] Envie o **sitemap** para o Google
- [ ] Configure o **Google Tag Manager** (se necessário)

### Divulgação

- [ ] Coloque o **link do site** na bio do Instagram
- [ ] Atualize o **WhatsApp Business** com o link do site
- [ ] Inclua o site no **cartão de visitas** e materiais impressos
- [ ] Adicione o site na **assinatura do email**
- [ ] Compartilhe o site nas **redes sociais**

## Manutenção contínua

- [ ] Revise o conteúdo a cada **3 meses**
- [ ] Atualize preços e serviços quando mudarem
- [ ] Publique artigos no blog **regularmente** (se tiver)
- [ ] Monitore as **visitas** pelo Google Analytics
- [ ] Verifique o **posicionamento** no Google periodicamente
- [ ] Renove o **domínio** antes do vencimento
- [ ] Faça **backups** regulares (se não forem automáticos)

## Perguntas frequentes (FAQ)

### Preciso completar tudo do checklist?

Os itens marcados são recomendações. Os essenciais são: conteúdo claro, design responsivo, contato visível e SEO básico. Os demais são diferenciais que melhoram os resultados.

### Com que frequência devo revisar o site?

No mínimo a cada 3 meses. Se seu negócio muda com frequência (novos serviços, novos preços), revise mais frequentemente.

### O Linkfl resolve todos os itens técnicos automaticamente?

A maioria sim. Design responsivo, SEO, SSL, hospedagem e velocidade são resolvidos automaticamente. Você só precisa se concentrar no conteúdo e na divulgação.
